#### 2.2.5 可执行性 (40/100) ❌ **关键问题**

**问题描述**:
测试代码无法被Maven Surefire识别和执行

**根本原因**:
1. **依赖缺失**: pom.xml中缺少junit-vintage-engine
2. **Provider不匹配**: Surefire使用JUnitPlatform Provider（JUnit 5），但测试代码是JUnit 4

**修复方案**:

**方案1: 添加JUnit Vintage Engine（推荐）**
```xml
<dependency>
  <groupId>org.junit.vintage</groupId>
  <artifactId>junit-vintage-engine</artifactId>
  <version>5.8.2</version>
  <scope>test</scope>
</dependency>
```

**方案2: 修改Surefire配置**
```xml
<plugin>
  <groupId>org.apache.maven.plugins</groupId>
  <artifactId>maven-surefire-plugin</artifactId>
  <configuration>
    <useSystemClassLoader>false</useSystemClassLoader>
  </configuration>
</plugin>
```

```java
// 改进方案
@Test
public void testAlertPoolThreadCount() {
    ExecutionContextExecutorService executor = ThreadUtils.executors;
    assertNotNull("Executor should not be null", executor);

    // 方案1: 使用反射获取线程池大小
    // 方案2: 通过并发任务验证容量
    int taskCount = 20;
    CountDownLatch latch = new CountDownLatch(taskCount);
    AtomicInteger completedCount = new AtomicInteger(0);

    for (int i = 0; i < taskCount; i++) {
        executor.execute(() -> {
            completedCount.incrementAndGet();
            latch.countDown();
        });
    }

    assertTrue("Should complete all tasks", latch.await(30, TimeUnit.SECONDS));
    assertEquals("All tasks should complete", taskCount, completedCount.get());
}

// 添加真实的并发测试
@Test
public void testConcurrentAlertSending() throws InterruptedException {
    int concurrentTasks = 20;
    CountDownLatch startLatch = new CountDownLatch(1);
    CountDownLatch endLatch = new CountDownLatch(concurrentTasks);
    AtomicInteger successCount = new AtomicInteger(0);

    // 创建20个并发任务
    for (int i = 0; i < concurrentTasks; i++) {
        final int taskId = i;
        ThreadUtils.executors.execute(() -> {
            try {
                startLatch.await(); // 等待所有线程就绪
                // 模拟alert发送
                Thread.sleep(100);
                successCount.incrementAndGet();
            } catch (InterruptedException e) {
                fail("Task should not be interrupted");
            } finally {
                endLatch.countDown();
            }
        });
    }

    long startTime = System.currentTimeMillis();
    startLatch.countDown(); // 同时启动所有任务
    boolean allCompleted = endLatch.await(60, TimeUnit.SECONDS);
    long duration = System.currentTimeMillis() - startTime;

    assertTrue("All tasks should complete", allCompleted);
    assertEquals("All tasks should succeed", concurrentTasks, successCount.get());
    assertTrue("Should complete within acceptable time", duration < 10000); // 10秒
}
```

```java
    // When: 执行清理（保留7天）

    // Then: 验证结果

    assertFalse("Directory should be deleted", Files.exists(Paths.get(TASK_DIR, "12345")));

```
